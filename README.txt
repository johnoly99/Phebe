This is the source files for the SeeMeCNC Phebe PCB Heated bed.

Some of the features of this board are:
dual voltage capabilities (12 or 24v)
hilbert curve traces for more surface area
special size for modified huxley's and other designs

This board was designed by Torben Larsen, for SeeMeCNC.

It is licensed under the all new SML

SML stands for SeeMeLicense.  That means, it's open source!  Take it and use it for whatever you'd like.  Just, if you get an idea from the design, credit the idea back to us, and we're all good here.
